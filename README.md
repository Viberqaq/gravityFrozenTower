这是一款来源自《江南百景图》的小游戏“垒鱼塔”的复刻  
- 纯前端项目，vibe coding实现
- 仅供交流学习、自娱自乐，**切勿用于盈利**

## 玩法
### 基础操作
支持手机触摸与电脑点击，同时电脑额外支持键盘输入
- 触摸/点击屏幕左/右侧时，小车会往左/右侧移动，也可使用A/D或←/→控制方向
- 触摸/点击下落按钮，可以以6倍速度加速下落，也可使用Z键或Q键
- 触摸/点击旋转按钮，可以顺时针旋转正处于下落状态的砖块90度，旋转后位置需要合法，也可使用X键或E键
- 触摸/点击循环按钮，可以重开本游戏，也可使用R键
- 触摸/点击X按钮，可以暂停本游戏，也可使用ESC键

### 游戏玩法
本游戏并非消除游戏，而是以叠高、堆积更多的砖块为目标的游戏。
#### 冰块
冰块分为两种，一种是由4个单位小方块组成的普通俄罗斯方块，有O/S/Z/L/J/I/T七种形状，还有一种由一个单位小方块组成的快速方块。
#### 冻结
1. 冰块和冰块、冰块和车厢、冰块和护栏之间都可以冻结，冻结条件是：
   - 两者的外轮廓至少有一条边部分重叠，所以以更平整地堆叠方块为目标吧
   - 保持这个重叠达到一定时间，如果重叠边没有来自1*1的快速方块的，这个冻结时间是15s，否则是2s
2. 冻结一定有两个对象参与，完成冻结后这两个对象视为一个对象。车厢/护栏是static的，所以任何和车厢/护栏直接或者间接冻结的都不会再受力的作用
3. 冻结后外观会发生变化，快速方块中间会出现白色菱形，普通方块会出现白色纹理
#### 惯性
这块做的比较粗糙，是通过施加外力来实现的。  
玩家的两次反方向操作如果在100ms内做完，那么所有未和车厢/护栏直接或间接冻结的方块均会在这两个反方向操作的前一个操作方向受到一个短时间的力，该力的作用点为方块所有在竖直方向上有投影的外轮廓。  
例如：玩家在100ms先后输入了 1.“左” 2.“右” 3.“左”， 那么1.“左” 2.“右”将会触发一个向左的力，2.“右” 3.“左”将会触发一个向右的力，形成两次晃动。  
再例如：如果车厢底部此时平躺着一个方块，距离车厢最左侧只有几个px了，那么可以间断快速输入“左右”来不断触发向左的力来让该方块顶到车厢最左侧。
